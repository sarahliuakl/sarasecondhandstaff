# Sara的二手售卖网站

> 新西兰奥克兰个人二手物品售卖平台

## 项目简介

这是一个专为新西兰地区打造的个人二手物品售卖网站，帮助Sara（居住在奥克兰北岸）将家中的优质二手物品出售给有需要的人。网站支持面交和邮寄两种交付方式，提供多种安全的支付选择。

## 主要功能

### 🛍️ 商品展示
- **产品分类**：电子产品、衣物、动漫周边、家电用品等
- **详细信息**：多角度图片、成色描述、规格参数、价格
- **搜索筛选**：按分类筛选、关键词搜索
- **库存管理**：实时显示商品可售状态

### 🛒 购物功能
- **购物车**：添加商品、数量管理、本地存储
- **产品详情**：完整的商品信息展示页面
- **交付方式**：当面交易（奥克兰地区）、邮寄（全新西兰）
- **支付选择**：ANZ银行转账、跨行转账、现金、移动支付

### 📞 客户服务
- **在线留言**：用户咨询和问题反馈
- **订单查询**：通过邮箱或电话查询订单状态
- **联系方式**：电话、邮箱直接联系
- **2小时回复**：承诺快速响应客户咨询

### 🔒 安全保障
- **风险提示**：跨行转账风险说明
- **安全交易**：当面交易地点保护隐私
- **订单追踪**：完整的订单状态管理
- **售后服务**：耐心的售后咨询支持

## 技术架构

### 后端技术
- **框架**：Flask 2.3.3
- **数据库**：SQLite + SQLAlchemy ORM
- **模板引擎**：Jinja2
- **表单处理**：WTForms
- **API接口**：RESTful API

### 前端技术
- **CSS框架**：Tailwind CSS
- **JavaScript**：原生JS
- **响应式设计**：移动端适配
- **本地存储**：LocalStorage购物车

### 数据库设计
- **产品表**：商品信息、分类、库存状态
- **订单表**：客户信息、商品清单、支付交付方式
- **留言表**：用户咨询、回复状态

## 快速开始

### 1. 环境要求
- Python 3.8+
- pip包管理器

### 2. 安装依赖
```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate  # Linux/Mac
# 或 venv\Scripts\activate  # Windows

# 安装依赖包
pip install -r requirements.txt
```

### 3. 初始化数据库
```bash
# 初始化数据库并导入示例数据
python init_db.py
```

### 4. 启动应用
```bash
# 启动Flask开发服务器
python app.py
```

访问 http://127.0.0.1:5000 查看网站
